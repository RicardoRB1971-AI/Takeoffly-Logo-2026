Introduction

TakeofFly is a professional desktop application built to perform construction and project takeoffs directly from PDF plans. It lets you place marks, measure dimensions and areas, organize items by CSI division, assign purchase orders and delivery notes, manage suppliers, generate reports, and visualize project progress — all in one place.

The application runs as a standalone desktop executable (Windows) and requires an active licensed account to operate. An internet connection is needed for authentication and for accessing the CSI database.

📐

Authentication & Login

TakeofFly uses a secure server-side authentication system. Every login is validated against a licensed account database.

Login Process

  1. Launch the TakeofFly application.
  2. Enter your registered Email address.
  3. Enter your Secret Key (provided at purchase — not your email password).
  4. Click Login. The system validates your credentials and device.
  5. Upon success the main workspace opens automatically.

Error Messages

MessageMeaningAction
Email not registered The entered email was not found in the system. Verify the email address or contact support.
Incorrect secret key Your email is correct but the secret key is wrong. Re-enter your secret key carefully. The email itself is correct.
License activated on another computer The license is already bound to a different machine. Contact riverablancoricardo@outlook.com to reset.
Tip: Your device is identified by CPU, hostname, and RAM — not by screen resolution. Changing monitors will not affect your authentication.

Project Setup

Starting a New Project

  1. Click New Project in the Main Bar.
  2. Enter a Project Name (required for saving).
  3. Set a Start Date.
  4. Click Load PDF and select your drawing file.
  5. Navigate pages using the page controls that appear once the PDF is loaded.

Company Logo

Click the logo area at the top of the sidebar to upload your company logo. It will appear on printed reports.

Sheet Identification

Each page of the PDF can be assigned a Sheet ID (e.g., A-100, S-201):

  • Auto — Automatically reads the sheet number from the drawing text.
  • Set Drawing ID Area — Draw a rectangle over the title block area to help the auto-detector.
  • Manual — Type the ID directly into the Sheet ID input field.
Tip: Always assign a Sheet ID before placing marks. It helps organize your report and allows marks to reference the correct drawing sheet.

Scale Calibration

Before measuring, you must calibrate the scale so TakeofFly converts pixels to real-world units (feet, meters, etc.).

How to Calibrate

  1. In the Property Bar, click Calibrate (Scale Tools section).
  2. Click on two known points on the drawing (e.g., the ends of a dimension line).
  3. Enter the real-world distance and select the unit.
  4. Click Save (💾 icon) to save the scale for reuse.
  5. Use the Saved scales dropdown and Apply to quickly set the same scale on other sheets.
Note: If you place marks without calibrating, check the "not necessary" checkbox. Measurements will be in pixels and the report will show an N.T.S. warning.

Found in the Tools section of the sidebar. These tools create marks that are counted and measured in the Report.

Rectangle ▭ Click and drag to draw a rectangle. Area (ft², m², etc.)

Subtract Tools

Subtract tools allow you to deduct areas from an existing polygon or rectangle measurement — for example, removing a void, column, or opening from a floor area.

ToolDescription
Subtract AreaSelect a parent area mark, then draw a polygon over the area to subtract. Double-click to finish.
Edit SubtractClick a subtraction polygon to reveal its vertices and reshape it.
Drag SubtractClick and drag a subtraction polygon to reposition it.
Clone SubtractClick a subtraction polygon to duplicate it on the same parent mark.
Remove SubtractClick a subtraction polygon to delete it from its parent mark.
Result: The parent mark's Total in the Report automatically reflects the net area after all subtractions.

Mark Details (Edit Properties)

The Mark Details panel opens when you use the Edit Properties tool or immediately after placing a new mark. It lets you classify and document each mark.

  • 1
    Division
    Select the CSI MasterFormat Division (e.g., 03 – Concrete, 08 – Openings). Required.
  • 2
    Category / Subdivision
    Select or type the CSI Subdivision code and description (e.g., 08 44 13 – Curtain Wall). You can also enter it manually.
  • 3
    MKD — Mark per Drawing
    The mark identifier as shown on the architectural drawing (e.g., FM-100, W-01). Used for cross-reference.
  • 4
    Comment (Detail)
    Free text describing the item (e.g., "Type A Window, Bronze Frame"). Use Obtain from drawing to drag a rectangle over text on the plan and auto-capture it.
  • 5
    Category
    Group marks by status or type. Default options always shown: Installed Purchased Quoted Received Shipped
    You can type any custom category and it will be saved for future use.
  • 6
    PO — Purchase Order
    Link this mark to a Purchase Order number (e.g., PO-2026-001). Appears in Report and Suppliers tables.
  • 7
    Delivery Note
    Link this mark to a vendor delivery note number (e.g., DN-2026-045). Appears in Report and Suppliers tables.
  • 8
    Reference Image (optional)
    Attach a product or detail image. Use Search Image in Google to find one online, paste a URL, or upload a local file. The image appears in the tooltip when you hover over the mark.

Hover Tooltip

Hovering your mouse over any mark on the canvas shows a dark tooltip displaying: Type, Division, Subdivision, MKD, Category, PO, Delivery Note, Comments, Value, Perimeter (if applicable). Fields are shown only if they contain data.

Repeat Properties

Check Repeat Properties in the Mark Details panel to automatically apply the same classification to the next mark you create — ideal for placing many identical items quickly.

Report

The Report view (accessible from the sidebar) provides a full summary of all marks in the project, organized in two tables.

Category Totals

Groups marks with the same Division + Subdivision + MKD + Comment + Category and shows the combined total. Columns:

Division  ·  Subdivision  ·  MKD  ·  Comment  ·  Category  ·  PO  ·  Delivery Note  ·  Type  ·  Color  ·  Total  ·  # Marks

Details

Lists every individual mark with its full data including location (sheet), type, and quantity.

Filter & Sort

  • Filter by — Sort both tables by Division, Subdivision, MKD, Comment, Category, PO, or Delivery Note.
  • Sort — A to Z or Z to A.
  • Category Filter — Click the funnel icon to show/hide specific categories in the report.

Export to Excel

Click Export to Excel at the bottom of the Report view to download a .xlsx file with both the Totals and Details tables.

Tip: Items marked with (R) have been revised through the Integration process. A Revised Items Notice section appears at the bottom of the report for traceability.
13

Chart

The Chart view provides a visual summary of project progress using an interactive donut chart based on the Category field of each mark.

How It Works

  1. Navigate to Chart from the sidebar.
  2. All categories found in the project are listed as filter chips at the top.
  3. Check or uncheck categories to include or exclude them from the chart.
  4. The donut chart updates instantly showing the proportion of marks per category.

What Is Shown

Donut Chart

Visual representation of mark counts by category. Center shows the total number of marks.

Breakdown Table

Lists each category with mark count, percentage, and a progress bar.

Summary Badges

Quick summary showing total marks and number of active categories.

Tip: Assign categories like Quoted, Purchased, Received, and Installed to your marks to see your procurement pipeline at a glance in the Chart view.

Suppliers

The Suppliers module allows you to maintain a directory of vendors and link project marks to them.

Adding a Supplier

  1. Go to the Suppliers view from the sidebar.
  2. Click Add Supplier.
  3. Fill in the supplier's name, contact information, and specialty.
  4. Save the supplier record.

Project Items Table

Each supplier profile includes a Project Items table showing all marks assigned to that supplier. Columns include: Division, Subdivision, MKD, Comment, Category, PO, Delivery Note, From Report, Type, and Total Quantity.

Filter & Sort

The Suppliers Project Items table supports the same Filter by and Sort options as the Report view, including filtering by PO and Delivery Note.

Export & Import

  • Export Suppliers — Saves the full supplier directory as a .json file.
  • Import Suppliers — Loads a previously exported supplier file and adds new entries.
15

Save, Recover & Integrate

Save

Click Save in the Main Bar to download the current project as a .json file. This file contains all marks, notes, scales, sheet IDs, and suppliers.

Important: The PDF file is not included in the saved project. Keep the original PDF file in a safe location so you can reload it when opening the project.

Recover

Click Recover and select a previously saved .json project file. Then load the corresponding PDF using Load PDF.

Integrate

Integrate merges a second team member's project file into the current one. This is used when multiple users are working on different sections of the same project.

  • Marks from the imported file are added to the current project.
  • Conflicting marks (based on ID) are flagged for review.
  • All integrations are logged and can be reviewed in the Integration Log.
Best Practice: Save your project frequently. Use descriptive project names that include the date or version for easy identification (e.g., "Aurora Tower – v2 – 2026-04-06").
